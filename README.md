## Setup Instructions

1. Clone the repository

2. Create a `.env` file in the root directory

3. Copy the content from `.env.example`:
   ```bash
   cp .env.example .env

4.Fill in your actual credentials in the .env file

5.Run the app :

Frontend:
In the terminal,

cd frontend
npm run dev

Backend:
In new terminal,

cd backend
npx nodemon
